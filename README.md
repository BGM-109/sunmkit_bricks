# sunmkit bundler

mason brick 저장소 입니다.
sunmkit 의 bundler 를 생성하는역활을합니다.

# how to use

```
git clone

mason get

mason ls 

mason add [brick]
```

# how to generate bundler 

```
mason bundle ./path/to/brick -o ./path/to/destination
```
